# Blind Project Approval - Supervisor Module

This ASP.NET Core MVC app implements the supervisor dashboard for blind review, including:

- Expertise area selection
- Blind proposal filtering using LINQ
- Blind review interface without student names
- Accept/Reject actions for supervisor decisions

## Run the app

1. Install the .NET SDK (recommended .NET 8).
2. Open the `BlindProjectApproval` folder.
3. Run:

```bash
dotnet run
```

4. Open the browser at `https://localhost:5001` or `http://localhost:5000`.
